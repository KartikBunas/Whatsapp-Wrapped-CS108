WhatsApp Group Wrapped -
In this project , our task is to create a set of codes which will analyze whatsapp chats and patterns of behaviour inside it and create a interactive slideshow . 
in the flow we will have -
1. chat.py - it will create chats
2. analyze.py - it will extract stats
3. app.js - it will show slideshow.
Now our set of codes will be in 4 parts 
first will be related to generating codes(chat.py)
second will be dealing with stats , parser will contain analyze.py
third part will contain in web information with java ,html and css files.
fourth part will contain results from first two parts and vacobulary.
